# 管理者の手順

---

## A-1. 新規ユーザーを登録する

**担当:** 管理者
**目的:** 学習者がログインできるアカウントを作成し、招待メールを送る

### 手順

**1. CSVファイルを準備する**

テキストエディタまたはExcelで以下の形式のCSVを作成する。

```
email,username,group
yamada@example.com,yamada.taro,
suzuki@example.com,suzuki.hanako,
tanaka@example.com,tanaka.admin,admin
```

| 列名 | 内容 | 必須 |
|------|------|------|
| email | メールアドレス（ログインIDになる） | ◎ |
| username | ユーザー名（半角英数字・ドット可） | ◎ |
| group | 権限グループ（管理者は `admin`、一般は空欄） | ー |

> **注意:** 同じメールアドレスが既に登録されている場合、そのレコードはスキップされる。

**2. 管理画面を開く**

メニューから「Cognitoユーザー」を選択する。

**3. CSVをアップロードする**

① ファイル選択エリアにCSVをドラッグ&ドロップ（またはクリックして選択）
② 内容をプレビューで確認する
③「アップロード」ボタンをクリックする

**4. 結果を確認する**

「成功N件・失敗M件」のサマリーが表示される。
失敗したレコードはエラー内容と行番号が一覧表示される。

**5. ユーザーへの案内**

登録されたユーザーには自動で招待メールが届く。
メールには仮パスワードが記載されており、初回ログイン時にパスワードの変更が求められる。

### 注意事項

- usernameは一度設定すると変更できない
- メールアドレスの重複はエラーになるが、処理は続行される（他の行は登録される）
- 一般ユーザーの `group` 列は空欄のまま（コース登録は受講時に自動で行われる）

## A-2. コースを新規開設する

**担当:** 管理者
**目的:** 学習者が受講できるコースをプラットフォームに追加する

### 手順

**1. CSVファイルを準備する**

```
fullname,shortname,categoryid,summary,visible,startdate,enddate
Webデザイン入門,web-intro-2026,1,HTMLとCSSの基礎を学ぶコースです,1,2026-04-01,2026-09-30
JavaScript基礎,js-basics-2026,1,JavaScriptの基礎を学ぶコースです,1,2026-04-01,
```

| 列名 | 内容 | 必須 |
|------|------|------|
| fullname | コースの正式名称 | ◎ |
| shortname | コースの短縮名（システム上の一意のID） | ◎ |
| categoryid | 所属カテゴリのID番号（Moodle管理画面で確認） | ◎ |
| summary | コースの説明文 | ー |
| visible | 1=公開、0=非公開 | ー |
| startdate | 開始日（YYYY-MM-DD形式） | ー |
| enddate | 終了日（YYYY-MM-DD形式） | ー |

> **shortnameの命名規則例:** `web-intro-2026`（カテゴリ-内容-年度）
> 一度設定したshortnameは変更しない。

**2. 管理画面を開く**

メニューから「コース作成」を選択する。

**3. CSVをアップロードする**

① CSVをドラッグ&ドロップ（またはクリックして選択）
② プレビューで内容を確認する
③「アップロード」ボタンをクリックする

**4. 結果を確認する**

「作成成功N件」が表示されたら完了。
Moodleの管理画面でもコースが追加されたことを確認できる。

**5. コースに教材を追加する**

コースの「箱」が作成された状態なので、続けて [A-6. コースに教材・アクティビティを追加する](#a-6-コースに教材アクティビティを追加する) を実施する。

### 注意事項

- categoryidはMoodle管理画面または「カテゴリ管理」ページで確認する
- shortnameが重複するとエラーになる
- startdate・enddateは省略可能（空欄の場合は制限なし）

## A-3. カテゴリを作成する

**担当:** 管理者
**目的:** コースを分類するカテゴリを追加する（コース作成の前に実施する）

### 手順

**1. CSVファイルを準備する**

```
name,parent,visible
Webデザイン,,1
HTMLとCSS,1,1
JavaScriptプログラミング,1,1
```

| 列名 | 内容 | 必須 |
|------|------|------|
| name | カテゴリ名 | ◎ |
| parent | 親カテゴリのID番号（トップレベルは空欄） | ー |
| visible | 1=表示、0=非表示 | ー |

> **parentの設定例:**
> 「Webデザイン（ID:1）」の下に「HTML&CSS」を作る場合 → parent=1

**2. 管理画面を開く**

メニューから「カテゴリ管理」を選択する。

**3. CSVをアップロードする**

① CSVをドラッグ&ドロップ
② プレビューで確認
③「アップロード」をクリック

**4. 作成されたカテゴリIDを確認する**

Moodle管理画面の「コース → コースとカテゴリを管理」でIDを確認する。
このIDは、コース作成CSVの `categoryid` 列で使用する。

## A-4. 教材（画像・HTMLページ）をアップロードする

**担当:** 管理者
**目的:** コース内で使用する画像やHTMLページをサーバーに保存し、URLを取得する

### 手順

**1. 管理画面を開く**

メニューから「コンテンツUP」を選択する。

**2. ファイルを選択する**

対応ファイル形式:

| 種類 | 形式 |
|------|------|
| 画像 | PNG / JPG / GIF / WebP / SVG |
| HTMLページ | .html / .htm |

> **上限:** 1ファイル 10MB まで

ファイルをドラッグ&ドロップ（またはクリックして選択）する。

**3. 保存先パスを確認・設定する**

ファイルを選択すると保存先パスが自動で入力される。

| ファイル種類 | 自動入力されるパス |
|------------|------------------|
| 画像 | `course-images/ファイル名` |
| HTML | `html-content/ファイル名` |

必要に応じてパスを変更する（例: `course-images/chapter01/fig1.png`）。

**4. アップロードする**

「アップロード」ボタンをクリックする。

**5. URLをコピーする**

アップロード完了後に表示されるURLをコピーする。
このURLをMoodleのコース教材内（画像の `src` や、リンク先）に貼り付けて使用する。

> **履歴:** アップロードしたファイルのURL一覧は画面下部の「履歴」に表示される。

### アップロードしたコンテンツの使い方

| 用途 | 使い方 |
|------|--------|
| コース内に画像を表示したい | MoodleのPageモジュールのHTML内に `<img src="コピーしたURL">` を挿入する |
| HTMLページをそのまま教材にしたい | MoodleのResourceモジュールにURLを設定する → 学習者にiframe表示される |

## A-6. コースに教材・アクティビティを追加する

**担当:** 管理者
**目的:** 作成済みのコースにセクション・ページ・動画・リンク等を追加する

> **この操作はMoodle管理画面で直接行う**（SPA管理画面では実施できない）

### Moodle管理画面での操作手順

**1. Moodleにログインする**

管理者アカウントでMoodleにアクセスする。

**2. 対象のコースを開く**

「コース → コースとカテゴリを管理」から対象コースを選択する。

**3. 編集モードを有効にする**

画面右上の「編集モード ON」をクリックする。

**4. モジュールを追加する**

「活動またはリソースを追加する」から追加するモジュールを選択する。

| モジュール名 | 用途 | 学習者画面での表示 |
|------------|------|-----------------|
| **Page（ページ）** | テキスト・HTML形式の教材 | iframeでHTML表示。目次が自動生成される |
| **Label（ラベル）** | セクションのタイトルや説明文 | テキストとして直接表示される |
| **URL** | 外部サイトへのリンク | iframeで外部サイトを表示 |
| **Resource（ファイル）** | 動画・PDFなどのファイル | 動画は再生プレーヤー表示。その他はダウンロードリンク |

**5. コンテンツを設定する（Pageモジュールの場合）**

① 「名前」にページタイトルを入力
② 「コンテンツ」にHTMLを入力する
　- 画像は A-4 でアップロードしたURLを `<img src="...">` で参照する
③「保存してコースに戻る」をクリックする

**6. アクティビティ完了条件を設定する（必要な場合）**

「完了とトラッキング」タブで完了条件を設定する。
これを設定することで学習者のマイページに進捗率（%）が表示される。

### Moodle管理画面で行う主な操作一覧

| 操作 | 手順の場所 |
|------|----------|
| セクション名の変更 | コース編集画面 |
| コースの公開・非公開 | コース設定 → 可視性 |
| バッジの作成・付与 | コース内バッジ設定 |
| 受講登録の手動追加・削除 | コース → 受講者 |
| 成績・完了状況の確認 | コース → レポート |

# 学習者の手順

---

## U-1. 初めてログインする

**担当:** 学習者
**前提:** 管理者から招待メールが届いていること

### 手順

**1. 招待メールを確認する**

管理者から「アカウント登録のお知らせ」メールが届く。
メールに記載された**仮パスワード**を控えておく。

**2. ログイン画面を開く**

ブラウザでシステムのURLにアクセスする。
ログイン画面が表示される。

**3. ログイン情報を入力する**

| 入力欄 | 入力内容 |
|--------|---------|
| メールアドレス | 招待メールの宛先アドレス |
| パスワード | メールに記載された仮パスワード |

「ログイン」ボタンをクリックする。

**4. 新しいパスワードを設定する**

初回ログイン時はパスワード変更画面が表示される。

| 入力欄 | 内容 |
|--------|------|
| 新しいパスワード | 自分で決めたパスワード |
| パスワード（確認） | 同じパスワードをもう一度 |

パスワードの要件:
- 8文字以上
- 大文字・小文字・数字・記号をそれぞれ1文字以上含む

「変更する」ボタンをクリックするとホーム画面に移動する。

**5. 次回以降のログイン**

メールアドレスと自分で設定したパスワードでログインする。

### 困ったとき

| 状況 | 対処方法 |
|------|---------|
| 仮パスワードが届かない | 迷惑メールフォルダを確認。なければ管理者に連絡 |
| パスワードを忘れた | ログイン画面の「パスワードを忘れた方はこちら」から再設定 |
| ログインできない | 管理者に連絡 |

## U-2. コースを探して受講を始める

**担当:** 学習者
**前提:** ログイン済みであること

### 手順

**1. コース一覧を開く**

ホーム画面またはメニューから「コース一覧」をクリックする。

**2. 受けたいコースを探す**

カテゴリごとにコースが表示されている。

| 探し方 | 操作 |
|--------|------|
| カテゴリから探す | カテゴリ名をクリック → そのカテゴリのコース一覧が表示される |
| 学習中のコースを確認する | 「学習中のコース」をクリック |

**3. コースを開く**

コース名をクリックする。

**4. 受講登録される**

コースを初めて開いたとき、自動的に受講登録される（操作不要）。
登録完了後、すぐにコンテンツが表示される。

## U-3. コンテンツを学習する

**担当:** 学習者

### コース画面の見方

```
┌──────────────────────────────────────────────────────┐
│  コース名                              ← 戻るボタン  │
├──────────────────┬───────────────────────────────────┤
│  目次             │  コンテンツ表示エリア              │
│  ・見出し1        │                                   │
│  ・見出し2        │  ページの内容・動画・リンクなど    │
│  ・見出し3        │                                   │
└──────────────────┴───────────────────────────────────┘
```

### コンテンツの種類と操作方法

| 種類 | 表示方法 | 操作 |
|------|---------|------|
| テキスト・HTML教材 | 画面内に直接表示される | スクロールして読む |
| 動画 | 画面内に動画プレーヤーが表示される | 再生ボタンをクリックして視聴 |
| 外部リンク | 画面内に外部サイトが表示される | そのまま閲覧 |
| ファイル（PDF等） | ダウンロードリンクが表示される | クリックしてダウンロード |

### 目次の使い方

画面左側に目次が表示される。
見出しをクリックすると、そのセクションまで自動でスクロールする。

### 学習の記録

アクティビティを完了すると、自動的に進捗率（%）が更新される。
進捗はマイページで確認できる。

---

## U-4. 学習状況を確認する

**担当:** 学習者

### マイページの見方

メニューから「マイページ」をクリックする。

**表示される情報:**

| セクション | 内容 |
|-----------|------|
| プロフィール | 氏名・キャリア目標 |
| 前回のつづき | 最後に学習したコース（「続きから学ぶ」ボタンで再開） |
| 学習中のコース | 受講登録済みコースの一覧と進捗率 |
| おすすめバッジ | 取得を目指せるバッジ一覧 |

**進捗率の見方:**

| 進捗率 | 意味 |
|--------|------|
| 0% | 受講登録済み、まだ未着手 |
| 1〜99% | 学習中（完了したアクティビティの割合） |
| 100% | コース内の全アクティビティ完了 |

**最終アクセスの表示例:**

- 「5分前」「3時間前」「2日前」のように相対時間で表示される

### バッジの確認

メニューから「バッジ」をクリックする。
取得済みのバッジはカラー表示、未取得はグレー表示になる。

## U-5. プロフィールを更新する

**担当:** 学習者

### 手順

**1. プロフィール編集画面を開く**

マイページの「プロフィール編集」ボタンをクリックする（または `/profile` に直接アクセス）。

**2. 情報を更新する**

| 項目 | 内容 |
|------|------|
| ニックネーム | 画面上に表示される名前 |
| キャリア目標 | 目指す職種・ゴール |
| 月間目標 | 今月の学習目標 |

**3. 保存する**

「保存」ボタンをクリックする。
保存完了のメッセージが表示されたら完了。

## ログアウトする

**担当:** 学習者・管理者共通

画面右上のユーザー名またはメニューから「ログアウト」をクリックする。
ログイン画面に戻ることを確認する。

> **注意:** 共有PCを使用している場合は必ずログアウトすること。

## よくある質問

| 質問 | 回答 |
|------|------|
| コースが表示されない | カテゴリが正しく設定されているか管理者に確認を依頼する |
| 動画が再生されない | ブラウザを最新版に更新する。それでも解決しない場合は管理者に連絡 |
| 進捗率が更新されない | ページをリロードする。Moodle側のアクティビティ完了条件が設定されているか確認 |
| パスワードを変更したい | マイページのプロフィール設定から変更できる |
| アップロードしたCSVでエラーが出る | エラー行の内容（重複・必須列の欠落等）をCSVで修正して再アップロード |
| S3アップロードが失敗する | ファイルサイズが10MB以下か確認する。形式が対応しているか確認する |
