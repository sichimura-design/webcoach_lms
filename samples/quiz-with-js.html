1HTMLで段落を表すタグはどれ？

✅正解！ <p> は "paragraph" の略で、段落を表すブロックレベル要素です。
❌不正解。正解は C の <p> です。"paragraph"（段落）の略です。

2CSSで文字色を赤にする正しい書き方は？

✅正解！ CSSで文字色を指定するプロパティは color です。
❌不正解。正解は C の color: red; です。font-color や text-color は存在しません。

3次のコードの表示結果は？

<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>
✅正解！ <ul> は "unordered list"（順序なしリスト）で、黒丸の箇条書きになります。
❌不正解。正解は B です。<ul> は順序なしリスト（箇条書き）です。番号付きにしたい場合は <ol> を使います。

4CSSの margin と padding の違いは？

✅正解！ margin は要素の外側、padding は要素の内側の余白です。
❌不正解。正解は B です。margin = 外側の余白、padding = 内側の余白です。

5画像を表示するHTMLタグの正しい書き方は？

✅正解！ 画像は <img> タグ + src 属性で指定します。alt はアクセシビリティ上も必須です。
❌不正解。正解は B です。画像は <img src="..." alt="..."> で表示します。href ではなく src です。
