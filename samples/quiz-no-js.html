選択肢を選んだら「こたえを見る」で正解を確認しましょう（全5問）

1HTMLで段落を表すタグはどれ？

こたえを見る
正解: C — <p>
<p> は "paragraph" の略で、段落を表すブロックレベル要素です。

2CSSで文字色を赤にする正しい書き方は？

こたえを見る
正解: C — color: red;
CSSで文字色を指定するプロパティは color です。font-color や text-color は存在しません。

3次のコードの表示結果は？

<ul>
  <li>りんご</li>
  <li>みかん</li>
</ul>
こたえを見る
正解: B — 箇条書き（黒丸付き）
<ul> は "unordered list"（順序なしリスト）で、デフォルトでは黒丸（・）の箇条書きになります。番号付きにしたい場合は <ol> を使います。

4CSSの margin と padding の違いは？

こたえを見る
正解: B — margin は外側、padding は内側
margin は要素の外側（他の要素との間）、padding は要素の内側（境界線とコンテンツの間）の余白です。

5画像を表示するHTMLタグの正しい書き方は？

こたえを見る
正解: B — <img src="photo.jpg" alt="写真">
画像は <img> タグで、src 属性に画像パスを指定します。alt 属性は画像が表示できない時の代替テキストで、アクセシビリティ上も必須です。
